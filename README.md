我们可以提供:

1. 完整的历史/实时的多市场数据
2. 完整的纯自研的流程构建, 可以对从数据/回测/分析/模拟/实盘/风控做完整的自定义的修改和兼容
3. 较为完整的可视化界面
4. 高性能的回测(tick/bar)分析模式
5. 视野的拓展/ 研报讨论
6. 私募通道的对接, 资金方的对接

1. 一个完整可实现的想法

3. 有较多的时间用于研究和开发
4. 有一定的热情
